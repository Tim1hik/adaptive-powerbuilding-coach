# Support The Project

Adaptive Powerbuilding Coach is a pet/open-source style project. Support is handled through Telegram, not email.

Public support channels:

- Telegram: `@tim1hik`
- DonatePay: configured through `NEXT_PUBLIC_DONATEPAY_URL`
- Crypto wallets: configured through public `NEXT_PUBLIC_CRYPTO_*` variables

## Mobile App Fund

The first public funding goal is `$100`. This is intended to start mobile app development and cover Apple Developer Program-related or publishing costs for the future iOS app.

Development starts when the goal reaches `$100`. The first 10 confirmed donors get lifetime free access to the future mobile app.

## Planned Public Benefits

- Apple Health integration.
- Automatic activity and sleep tracking.
- Push reminders.
- Offline workout diary.
- Deeper training adaptation.
- Faster workout logging.
- Future premium analytics.

Some mobile features are intentionally kept secret until the official mobile app development announcement.
